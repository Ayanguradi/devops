### Data Transfer Tool:--

* wget:It can download the file from given URL and protocols which it interacts is http,https,ftp
* curl:It can download or upload the file from given URL and interact with API and it supports variety of protocols
* But this tool gets dependencies error while downloading software if dependencies are not there. Then this can be solved using package manager like yum



### Package Managers:--

* centOS: rpm,yum,dnf

  -yum(high level tool) is better than rpm(low level tool) because it has more features than rpm

   like dependency resolution and automatic updates

  -dnf is successor to yum its newer and better which improves performance, memory management,dependency resolution

   and includes features like parallel downloading and modular repositories.

* Ubuntu: dpkg, apt

   -dpkg is similar to rpm

   -apt is similar to yum
