### Unix:--

-Its a closed source OS where we cant access its source code

-It is used to design a specific hardware architecture like servers



### GNU:--

-It stands for gnu's not unix

-Its main purpose is to build a OS like Unix but it should be open source



### LINUX:--

-Its built on the concepts of unix but its not unix

-Its not entirely an OS its a kernel

-Its open source

-It is used in various hardware devices

-Applications: devops, Cyber security, servers



### Kernel:--

-Its a interface or intermediator between the hardware and software.

-Its a manager who manages resource which is required by software.

-There are different types of kernel which is used by different OS

  Linux is kernel on which Linux distros are built like ubuntu, centos, kali-Linux and android

  windows-NT is a kernel on which Windows is built

 

### Collab of Linux and GNU:--

-Linux uses some tools of GNU kernel

-Based on both collaboration most of the famous OS are built

-Linux kernel + GNU utilities = complete, opensource unix like OS
